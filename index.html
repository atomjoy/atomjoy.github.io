Frontend

Strony internetowe tworzę w Vue 3, JavaScript, Html i Css. Html styluję responsywnie (RWD) w Css a do animacji na stronie www używam biblotek Gsap3 lub AnimateCss. Grafikę i strony www projektuję w Gimp a svg i loga w Inscape.

Backend
